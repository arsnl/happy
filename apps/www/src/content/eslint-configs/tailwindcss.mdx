## Usage

Optional configuration if [Tailwind CSS](https://tailwindcss.com/) is used in your project.

```js
module.exports = {
  extends: ["peppy", "peppy/tailwindcss"],
};
```

## Dependencies

The following dependencies are included in this configuration.
